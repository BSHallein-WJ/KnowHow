# TechZeichner Pro - Interaktiver Lerntrainer

Ein moderner, webbasierter Lerntrainer für angehende Technische Zeichner und Metalltechnik-Auszubildende. Diese App wurde entwickelt, um Schülern das Lernen von Fachwissen in verschiedenen Modulen (MTE, AMA, CTZ, LAÜ, CNC) zu erleichtern.

## 🚀 Funktionen

- **Modulares Lernen**: Verschiedene Themengebiete (z.B. Messtechnik, Fertigungstechnik, Schnittdarstellungen).
- **Zufalls-Abfrage**: Fragen werden in zufälliger Reihenfolge gestellt, um den Lerneffekt zu maximieren.
- **Favoriten-System**: Markiere schwierige Fragen und übe sie gezielt in einer eigenen Kategorie.
- **Bild-Unterstützung**: Unterstützung für Fragen mit bis zu drei Bildern und interaktiver Zoom-Funktion für Details.
- **Dark & Light Mode**: Einstellbares Design für optimales Lernen zu jeder Tageszeit.
- **Datenschutz**: Alle Fortschritte werden lokal im Browser gespeichert (`localStorage`). Es erfolgt keine Datenübertragung an Server.

## 🛠️ Installation & Nutzung

Die App ist eine reine Client-Anwendung (HTML/JS) und benötigt keinen Server.

### Option 1: Lokal nutzen
1. Lade dieses Repository herunter.
2. Öffne die Datei `index.html` in einem modernen Webbrowser (Chrome, Firefox, Safari, Edge).

### Option 2: GitHub Pages (Online)
Du kannst diese App auch direkt online über den von GitHub bereitgestellten Link aufrufen (falls GitHub Pages aktiviert ist).

## 📂 Struktur des Projekts

- `index.html`: Die Hauptanwendung.
- `*.json`: Dateien mit Fragen und Antworten (z.B. `ctz.json`).
- `Media/`: Ordner für begleitendes Bildmaterial zu den Fragen.
- `impressum.html`: Gesetzliche Pflichtangaben.
- `datenschutz.html`: Datenschutzerklärung gemäß DSGVO.
