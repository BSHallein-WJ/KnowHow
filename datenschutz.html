Datenschutzerklärung

1. Datenschutz auf einen Blick

Die Nutzung dieser App ist ohne Angabe personenbezogener Daten möglich. Die App wurde so konzipiert, dass sie vollständig ohne externe Abhängigkeiten funktioniert und keine Daten an Dritte überträgt.

2. Verantwortliche Stelle

Die verantwortliche Stelle für die Datenverarbeitung ist der im Impressum genannte Betreiber dieser Website.

3. Host-Provider und Server-Log-Dateien

Hosting über GitHub

Diese Website wird über den Dienst „GitHub Pages“ der Firma GitHub, Inc. (Anschrift: 88 Colin P Kelly Jr St, San Francisco, CA 94107, USA) bereitgestellt. Wenn Sie diese Seite aufrufen, werden technische Daten (IP-Adresse, Zeitpunkt des Zugriffs, Browsertyp) von GitHub verarbeitet, um die Seite auszuliefern.

Die Datenübermittlung in die USA wird auf die **Standardvertragsklauseln der EU-Kommission** gestützt. Die Rechtsgrundlage für die Verarbeitung ist unser berechtigtes Interesse an einer stabilen und sicheren Bereitstellung der App (Art. 6 Abs. 1 lit. f DSGVO).

Die Server-Log-Dateien werden von GitHub gemäß deren Richtlinien in der Regel für 30 Tage gespeichert und anschließend automatisch gelöscht. Weitere Informationen finden Sie unter: GitHub Privacy Statement.

Local Storage (Lokal auf Ihrem Gerät)

Diese App nutzt den „Local Storage“ Ihres Browsers, um Ihre Lernfortschritte lokal auf Ihrem Gerät zu speichern. Dies ist technisch erforderlich, um die Kernfunktionen der App (das Lernen und Speichern von Fortschritten) bereitzustellen (Art. 6 Abs. 1 lit. f DSGVO). Diese Daten verlassen Ihr Gerät zu keinem Zeitpunkt.

Keine Tracking-Tools

Diese Website verwendet keine Analyse-Tools, keine Marketing-Cookies und keine externen Ressourcen von Drittsystemen (außer der technischen Auslieferung durch GitHub).
